Limite de 2 joueurs pour une partie.

Timer par tour de 15 secondes.
(Possibilité d'ajouter modification des options d'une partie lors de la création).

'Créer une partie' et 'Rejoindre une partie' sont 2 options différentes
	Le créateur d'une partie ne rejoins pas automatiquement celle-ci

Suite requete de creation du jeu, reponse du serveur => nouvelle instance de jeu :

new MemoryGame = {
	cards: 18,
	colums: 6,
	rows: 3,
	players: [
		{name: 'joueurA', color: 'rouge'},
		{name: 'joueurB', color: 'bleu'}
	]
}

game: {
	id: gameID,
	players: [
		{nickname: 'Winner1648', client: 'clientID', color: 'red', wins: 0, currentScore: '5'},	
		{nickname: 'Loser25', client: 'clientID', color: 'blue', wins: 1, currentScore: '2'}
	],
}

Reponse envoyée à tous les joueurs présents.

Requete joueur en partie :
	{
	clientId: 'currentClientId',
	gameId: 'theCurrentGame'sId',
	cardsSelected: [
		'banana',
		'apple'
		]
	}

Reponse serveur :
	{
	method: 'update',
	game: {
		id: gameId,
		state: [
			['banana', 'returned', 'notFounded', ''],
			['apple', 'returned', 'notFounded', ''],
			['peach', 'show', 'founded', 'Winner1648'],
			['strawberry', 'hide', 'notFounded', ''],
			['melon', 'hide', 'notFounded', ''],
			['peach', 'show', 'founded', 'Winner1648'],
			['cherry', 'hide', 'notFounded', ''],
			...
			]
		}
	}
